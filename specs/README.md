# Specs — 스펙 인덱스

스펙 주도 개발의 스펙 목록. 스펙은 상태별 디렉터리로 분류하며, 상태 전환은 파일 이동으로 표현한다.

```
specs/
  not-started/   # 작성됐지만 구현 전
  in-progress/   # 구현 중
  completed/     # 구현·검토 완료
```

이 표는 **인덱스**다. 스펙을 추가·이동·삭제하면 즉시 맞춘다. 값은 각 스펙 frontmatter와 일치시킨다.
